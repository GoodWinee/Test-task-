# API для вопросов и ответов

## Описание
Это REST API для создания и управления вопросами и ответами. Проект реализован с помощью Django и Django REST Framework.
Позволяет пользователям создавать вопросы, отвечать на них и просматривать содержимое.

## Стек технологий
- Django 4.x
- Django REST Framework
- Docker и Docker Compose

## Как запустить проект

#### С использованием Docker и Docker Compose

1. Убедитесь, что Docker установлен на вашей машине.
2. В терминале перейдите в папку проекта и выполните команду:

```bash
docker-compose up --build
